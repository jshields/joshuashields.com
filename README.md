# joshuashields.com

Website of Joshua Shields.

## Usage:

```
nvm install 22.3.0
nvm use 22.3.0
npm install
npm install -g grunt-cli

# see Gruntfile.js
grunt lint
grunt deploy
```

Lint:
```
npx stylelint "**/*.scss"
```
